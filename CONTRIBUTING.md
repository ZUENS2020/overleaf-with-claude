# Contributing

This is a fork of [Overleaf Community Edition](https://github.com/overleaf/overleaf) with a
Claude Code AI assistant. Contributions are welcome.

## Reporting Issues

Please [open an issue](https://github.com/ZUENS2020/overleaf-with-claude/issues) with:

- What you expected to happen
- What actually happened
- Steps to reproduce
- Screenshots for UI issues

For issues specific to the upstream Overleaf Community Edition (not our AI assistant changes),
please report to [overleaf/overleaf](https://github.com/overleaf/overleaf/issues).

## Pull Requests

1. Fork the repo
2. Create a feature branch
3. Make your changes
4. Open a PR against `main`

## Upstream

To stay in sync with upstream Overleaf CE:

```bash
git remote add upstream https://github.com/overleaf/overleaf.git
git fetch upstream
git merge upstream/main
```
